The Python file GP_DRT is the source code file provided by the authors of the paper referenced below. The code was obtained from https://github.com/ciuccislab/GP-DRT/tree/master/tutorials on July 2, 2020.

Additional files were written by Jake Huang to apply the GP DRT model to various spectra.

Reference:
Liu, J., & Ciucci, F. (2020). The Gaussian process distribution of relaxation times: A machine learning tool for the analysis and prediction of electrochemical impedance spectroscopy data. Electrochimica Acta, 331, 135316. https://doi.org/10.1016/j.electacta.2019.135316
