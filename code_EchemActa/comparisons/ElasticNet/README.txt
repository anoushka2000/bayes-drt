The MATLAB files sms_DRT.m and cal_Basis.m are the source code files provided by the authors of the paper referenced below. The code was obtained from https://github.com/nonmin/EIS-DRT on July 2, 2020.

Additional files were written by Jake Huang to apply the Elastic Net DRT model to various spectra.

Reference:
Li, X., Ahmadi, M., Collins, L., & Kalinin, S. V. (2019). Deconvolving distribution of relaxation times, resistances and inductance from electrochemical impedance spectroscopy via statistical model selection: Exploiting structural-sparsity regularization and data-driven parameter tuning.
